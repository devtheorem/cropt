Croppie

Croppie is a fast, easy to use image cropping plugin with tons of configuration options!

View on Github

The Basics

<div class="demo"></div>
<script>
    var c = new Croppie(document.querySelector('.demo'));
    c.bind('demo/demo-1.jpg');
</script>

Installation

NPM:

npm install croppie

Then add the following elements to your page:

<link rel="stylesheet" href="croppie.css" />
<script src="croppie.js"></script>

Usage

You can initialize Croppie with the following code:

var c = new Croppie(document.getElementById('item'), opts);
// call a method
c.method(args);

Documentation

Options

  • boundaryobject

    The outer container of the cropper

    Default will default to the size of the container
  • customClassstring

    A class of your choosing to add to the container to add custom styles to your croppie

    Default''
  • enableResizeboolean

    Enable or disable support for resizing the viewport area.

    Default false
  • enableZoomboolean

    Enable zooming functionality. If set to false - scrolling and pinching would not zoom.

    Defaulttrue
  • mouseWheelZoombool/string

    Enable or disable the ability to use the mouse wheel to zoom in and out on a croppie instance. If 'ctrl' is passed mouse wheel will only work while control keyboard is pressed

    Defaulttrue
  • showZoomerboolean

    Hide or Show the zoom slider

    Defaulttrue
  • viewportobject

    The inner container of the coppie. The visible part of the image

    Default { width: 100, height: 100, type: 'square' }

    Valid values:'square' | 'circle'

Methods

  • getPoints()object

    Get the crop points.

  • bind(src, zoom)Promise

    Bind an image from an src string. Returns a Promise which resolves when the image has been loaded and state is initialized.

    Parameters
    • src URL to image
    • zoom A number between 0 and 1, or null
    • Default:null
  • destroy()

    Destroy a croppie instance and remove it from the DOM

  • refresh()

    Recalculate points for the croppie image. Necessary if croppie instance was bound to a hidden element.

  • toCanvas(size)Promise

    Returns a Promise resolving to an HTMLCanvasElement object for the cropped image.

    • size If specified, the image will be scaled with its longest side set to size.
    • Default:null (the image will not be scaled)
  • toBlob(size, type, quality)Promise

    Returns a Promise resolving to a Blob object for the cropped image.

    • size If specified, the image will be scaled with its longest side set to size.
    • Default:null (the image will not be scaled)
    • type A string containing the image format.
    • Default:"image/webp"
    • Valid Values:"image/webp" | "image/png" | "image/jpeg"
    • quality A number between 0 and 1 indicating image quality.
    • Default:1
  • setZoom(value)

    Set the zoom of a Croppie instance. The value passed in is still restricted to the min/max set by Croppie.

    • value a floating point to scale the image within the croppie. Must be between a min and max value set by croppie.

Demos

Basic Example 
var basicEl = document.getElementById('demo-basic');
var basic = new Croppie(basicEl, {
    viewport: {
        width: 150,
        height: 200
    }
});
basic.bind('demo/cat.jpg', 0.5);
//on button click
basic.toCanvas(300).then((canvas) => {
    let src = canvas.toDataURL();
    // src contains a data URI which can
    // be set as the src of an image element.
});
Resizer Example 
var el = document.getElementById('resizer-demo');
var resize = new Croppie(el, {
    viewport: { width: 150, height: 150 },
    boundary: { width: 300, height: 300 },
    showZoomer: false,
    enableResize: true,
    mouseWheelZoom: 'ctrl'
});
resize.bind('demo/demo-2.jpg', 0);
//on button click
basic.toBlob(300, "image/jpeg", 0.9).then((blob) => {
    // do something with cropped blob
});
Upload Example 
var uploadEl = document.getElementById('upload-demo');
var uploadCrop = new Croppie(uploadEl, {
    viewport: {
        width: 200,
        height: 200,
        type: 'circle'
    },
    boundary: {
        width: 300,
        height: 300
    }
});

Hidden Example

When binding a croppie element that isn't visible, i.e., in a modal - you'll need to call refresh on your croppie instance, to indicate to croppie that the position has changed and it needs to recalculate its points.

hiddenCrop.refresh();

Important Notes

Image Hosting & Cross Origin Errors

Croppie uses canvas.drawImage(...) to manipulate images. Thus, images must obey the CORS policy. More info can be found here.

Visibility and Binding

Croppie is dependent on its container being visible when the bind method is called. This can be an issue when your croppie component is inside a modal that isn't shown. Let's take the bootstrap modal for example..

var cropEl = document.getElementById('my-croppie');
var myCroppie = new Croppie(cropEl, opts);
document.getElementById('my-modal').addEventListener('shown.bs.modal', function () {
    myCroppie.bind(url);
});

If you are having issues getting the correct crop result, and your croppie instance is shown inside of a modal, try taking your croppie out of the modal and see if your issues persist. If they don't, then make sure that your bind method is called after the modal is done animating.

Browser Support

Croppie is tested in the following browsers:

  • Firefox
  • Chrome
  • Edge
  • Safari

Croppie should also work in any other modern browser using an engine based on Gecko, Webkit, or Chromium.
