Cropt

Cropt is a modern, easy-to-use image cropper written in JavaScript.

View on Github

Demos

Basic Example 
var basicEl = document.getElementById('demo-basic');
var basic = new Cropt(basicEl, {
    viewport: {
        width: 150,
        height: 200
    }
});
basic.bind('demo/toucan.jpg', 0.25);
//on button click
basic.toCanvas(350).then((canvas) => {
    let src = canvas.toDataURL();
    // src contains a data URI which can
    // be set as the src of an image element.
});
Upload Example 
var uploadEl = document.getElementById('upload-demo');

Hidden Example

When binding to an element that isn't visible (e.g. in a modal), you'll need to call refresh on your cropt instance to indicate that the position has changed and points need to be recalculated.

hiddenCrop.refresh();
